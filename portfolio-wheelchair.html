Project Title: Hand Gesture Controlled Wheelchair for Disabled people

Project Information:

Associated With: EEE 318 Control System Laboratory

Project Date: September, 2023

Skills: Project Management 路 Arduino 路 Hardware Installation 路 Control Circuits 路 Managing Large Scale Projects

Project Details:

Details: In this project, we were set out to design & build a cost-friendly wheelchair that can be controlled via hand gestures. According to the preliminary report of NATIONAL SURVEY ON PERSONS WITH DISABILITIES 2021 conducted by BANGLADESH BUREAU OF STATISTICS, 2.8% people have physical disability, on which 3.28% is for male and 2.32% is for female. A large portion of the people with physical disabilities face problems regarding their movement. In this project we ensure, the person only needs to move his/ her one hand to make gestures in order to control this wheelchair.
